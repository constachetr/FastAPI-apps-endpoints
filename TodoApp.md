1 - TodoApp is a simple and efficient API-based application built with FastAPI and SQLite3. This application demonstrates essential CRUD (Create, Read, Update, Delete) operations with user authentication and authorization functionalities, making it a great starting point for scalable backend applications.
Features

User Management:

User registration with secure password hashing.

User login with JWT-based authentication.

Role-based access control for authorized actions.

Todo Management:

Create, Read, Update, and Delete Todos.

Associate todos with specific users.

Ensure that users can only manage their own todos.

Security:

Passwords are hashed using industry-standard algorithms (via Passlib).

JWT tokens are used for user authentication.

Database:

Lightweight and efficient data storage with SQLite3.
